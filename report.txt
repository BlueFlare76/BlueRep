Обзор проекта:

•Работа с биологическими последовательностями (ДНК, белки);
•Анализ геномных данных;
•Визуализация данных;
•Моделирование популяций.

Структура проекта:
Лабораторная работа 1 
Цель: Моделирование роста популяции кроликов

Функционал:
•Реализация модели роста популяции с параметрами n (месяцы) и k (коэффициент размножения)
•Использование pandas для хранения и обработки данных
•Ограничение входных параметров (n ≤ 40, k ≤ 5)
•Пример использования: quan_rab(n=13, k=5)

Лабораторная работа 2 (Lab2.py)
Цель: Перевод ДНК последовательностей в белковые

Функционал:
•Чтение FASTA файла с ДНК последовательностью
•Поиск открытых рамок считывания (ORF)
•Перевод в белковые последовательности с учётом прямых и обратных рамок
•Учёт стартовых (M) и стоп-кодонов

Лабораторная работа 3
Часть 1 
Цель: Визуализация данных Iris

Функционал:
•Загрузка набора данных Iris из sklearn
•Создание DataFrame с помощью pandas
•Построение scatter plot для ширины чашелистика и длины лепестка
•Цветовая дифференциация по классам

Часть 2
Цель: Анализ данных о расходе воды в Ниле

Функционал:
•Загрузка исторических данных о расходе воды
•Построение временного ряда (1871-1970)
•Настройка визуализации (подписи, легенда, сетка)

Лабораторная работа 4
Часть 1 
Цель: Получение геномных данных из NCBI

Функционал:
•Поиск последовательностей для Brassica oleracea и Solanum lycopersicum
vЗагрузка полных кодирующих последовательностей (CDS)
•Сохранение результатов в GenBank файл

Часть 2
Цель: Анализ GC-состава последовательностей

Функционал:
•Чтение GenBank файла
•Расчёт GC-состава для каждой последовательности
•Сортировка и вывод результатов

Часть 3
Цель: Извлечение белковых последовательностей

Функционал:
•Анализ features в GenBank записях
•Поиск mRNA и CDS features
•Перевод нуклеотидных последовательностей в аминокислотные


Основные используемые библиотеки:
•Biopython (для работы с биологическими последовательностями)
•pandas (для обработки табличных данных)
•matplotlib (для визуализации)
•scikit-learn (для доступа к набору данных Iris)
•statsmodels (для доступа к данным о Ниле)
